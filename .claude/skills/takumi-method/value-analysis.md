# Phase 3: 価値分析モデル(value-analysis)

## 価値分析モデル

### 価値分析モデルとは

ステークホルダーが抱くニーズを明確化し、企画が提供する**表層的価値**（見える価値<機能性、見た目、デザイン、価格>）を分析し、可視化するモデル。「情」のモデル。

### 価値分析モデルの構成要素

#### 1. 価値記述（Value Description）
- ステークホルダーごとの**具体的な価値**。企画が成功した時の未来の価値。
- 具体的な書き方は、「価値記述の書き方」に記載されている

#### 2. 目的（Objective）
- 複数の価値を帰納法でグループ化したもの
- コンセプトをさらに具体化し
- 価値デザインモデルのビジョン・コンセプトを実現するための標的
- 企画チーム（プロジェクトチーム）の目線から表現する
- パーパス(purpose)とは異なる概念

## 目的

### 目的とは

ビジョンが最終的に目指す理想の状態であるのに対し、目的はその実現に向けて進む方向を定める概念。

ここでいう「目的」は、パーパス経営で用いられるPurpose（存在意義）ではない。

本稿における目的とは、的を意味するObjectiveを指す。すなわち、価値の実現に向けて設定する具体的な到達点であり、施策や数値目標を方向づける上位概念。

匠Thinkの言葉「目的だけでは足りない。価値を問え。」

「目的」は、多くの場合、企業側の視点や思惑で設定されがちである。重要なのは、企業の都合ではなく、「ステークホルダーが求めている体験や恩恵（価値）は何か」を問い直すこと。

### 目的の作成例

- 地域経済の活性化
- 持続可能な生産・販売ネットワークの構築
- 組織全体の企画力の底上げ
- 販売施策の組み合わせによる売上向上
- 業務効率化による費用削減

### 目的の悪い例

いずれも企業側の都合に寄った抽象的な表現であり、誰にどのような価値をもたらすのかにつながらない。

- 売上の向上
- 費用の削減
- 社員満足度の向上

### 目的の作成方法

- 価値分析モデルの全ての価値記述を対象とする
- 8個以上の目的を出力する
- 各目的に対して説明文を作成する
- 関連する価値の名前を特定する
- 関連するコンセプトを特定する

**作成手順**:
1. 全ての価値記述を確認する
2. 類似する価値記述をグループ化する
3. 各グループに対して目的を命名する
4. 各目的に対して説明文を作成する
5. 関連する価値の名前を特定する
6. 関連するコンセプトを特定する


### 目的と価値記述の関係

```
価値記述A ─┐
価値記述B ─┼→ 目的1（グループ化）
価値記述C ─┘

価値記述D ─┐
価値記述E ─┼→ 目的2（グループ化）
価値記述F ─┘
```

## ステークホルダー
- ステークホルダーモデル(`output/stakeholder.tsv`)に出力されたステークホルダーをリストアップして配置する
- 重要度が「高」「中」のステークホルダーのみを対象とする

## 価値記述

### 価値記述の書き方

#### 基本形式

自然言語で「シチュエーション」「価値を実現する手段」+「価値」の一文で記述する。
- 「価値」の部分は、必ず **「〜で嬉しい」** という形で表現する
- 価値記述には、端的に意味を表す名前をつける

#### 価値記述の例

- **価値記述名**: アイデア発信
- **価値記述**: 「自分がデザインした家具を誰かに使ってもらいたい時に、デザインした家具をマーケットプレイスで公開することで、アイデアを世界に発信できて嬉しい」
  - シチュエーション:「自分がデザインした家具を誰かに使ってもらいたい時に、」
  - 価値を実現する手段: 「デザインした家具をマーケットプレイスで公開することで、」
  - 価値: 「アイデアを世界に発信できて嬉しい」

##### 良い例
- 「月末の経理作業時に、業務フローが簡潔なので、作業工数が3日から1日に短縮されて嬉しい」
- 「営業報告レポートの作成時に、使いやすい画面なので、入力ミスによる修正作業がなくなって嬉しい」
- 「経営状況を確認したい時に、経営状況のレポートがリアルタイムで出力されるので、タイムリーに把握できて嬉しい」

##### 悪い例
- 「売上が増えて嬉しい」（具体性がない）
- 「効率化」（何がどう効率化されるか不明）
- 「安心」（Phase 2の深層的価値と混同）

### 価値記述の対象と並び順

#### 対象ステークホルダー
- 重要度が「高」「中」のステークホルダーのみを対象とする
- 重要度「高」のステークホルダーは、できるだけ多くの価値記述を列挙する

#### 並び順
- ステークホルダーにとって最もインパクトのある価値を先頭に置く
- その後は価値が高い順に並べる
- 全体の順序: 重要度「高」→「中」の順

### 価値の種類

価値記述を考える際に以下を参考にすること：

#### 1. 機能的価値
- 時間の節約、簡素化、収益確保、リスク低減、整理・整頓
- 統合する、つなぐ、労力の軽減、面倒の回避、コスト削減
- 品質、バラエティ、感覚訴求、情報提供

#### 2. 感情的価値
- 不安の軽減、自分へのほうび、懐かしさ、デザインや美観
- 象徴性、健康、癒やし、娯楽、魅力、つながりの提供

#### 3. 人生に変化を与える価値
- 希望の創出、自己実現、モチベーション、資産継承、帰属・縁

#### 4. 社会に影響を与える価値
- 自己超越
