## 説明すべき内容

1. **匠Methodとは**
   - 「価値」を起点にビジネスやサービスを創造・デザインする体系的な企画手法
   - 「知・情・意」の統合

2. **4つのフェーズの概要**
   - Phase 1: ステークホルダーモデル（Who）
   - Phase 2: 価値デザインモデル（Why - 本質/シーズ）
   - Phase 3: 価値分析モデル（Why - 顕在/ニーズ）
   - Phase 4: 要求分析ツリー（What/How）

3. **フェーズの流れと関係性**
   - 各フェーズの順序
   - Phase 2とPhase 3の入れ替え可能性
   - 前フェーズとの整合性の重要性

4. **「情」と「意」の違い**
   - 「情」: 顧客が感じる価値（ニーズ）
   - 「意」: 提供する側が持つ価値（シーズ）

5. **価値創造サイクル**
   - 価値のデザインから具体的な活動計画までの一貫した流れ

6. **品質基準（共通）**
   - 各フェーズの品質チェックポイント

7. **成果物一覧**
   - TSVファイルとDrawIOファイルの対応
