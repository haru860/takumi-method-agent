# 重要な注意事項

## 入力ファイルの優先

**生成する内容は、必ず入力ファイル（`input/idea.md`、`output/value-design.tsv`）の実際の内容に基づくこと。**

- 以前のセッションの情報や、他の企画の内容を使用しない
- サンプルデータや例示の固有名詞（「プロアスリート」「Cloud Base」等）は使用しない
- 入力ファイルに記載されていない企画や意味を生成してはならない

# 実行手順

## 手順1: 知識の読み込み

`Read` で `.claude/skills/takumi-method/value-design.md` を読み込む。

## 手順2: 入力ファイルの読み込み

1. `Read` で `input/idea.md` を読み込む
2. `Read` で `output/value-design.tsv` を読み込む
   - 存在しない場合はユーザーにPhase 2-3の完了を求める
   - ビジョン、コンセプト、言葉を確認する

## 手順3: 意味の作成

手順1で読み込んだ知識に従い、ビジョン・コンセプト・言葉を表現する以下を作成する：

- **意味**: 社会的意義
  - この企画が社会にもたらす価値
  - なぜこの企画が必要なのか
  - 価値デザインモデル全体を文章で説明する

作成結果として、意味を明確にする。
